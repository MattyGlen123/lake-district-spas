## Spa Access Models

**This is a key competitive advantage of the site - use this information when relevant.**

### Model 1: Spa Access Included (Free for All Guests)

These properties include full spa access with every room booking:
- Armathwaite Hall
- Another Place
- Brimstone Hotel
- Daffodil Hotel
- Ambleside Salutation
- Lakeside Hotel
- Rothay Garden
- Beech Hill Hotel
- Macdonald Old England
- North Lakes Hotel
- Whitewater Hotel

### Model 2: Tiered/Partial Access

Access depends on room type or package:

**Lodore Falls:**
- Suites get 2 hours free spa access per night stayed
- Standard rooms must pay £35 per person for 2-hour spa access
- Book spa slots after confirming room booking

**Swan Hotel Newby Bridge:**
- Splendid rooms and above include free spa access
- Standard rooms must purchase day pass separately
- 18+ only (16-17 with parent/guardian)

**Grange Hotel:**
- Some overnight packages include spa access
- Other guests pay additional charge for spa access
- Spa day packages available combining treatments with facilities

### Model 3: Separate Charge for All Guests

Everyone pays for spa access regardless of room type:

**Low Wood Bay:**
- Spa access NOT included with any room type
- All guests must book spa sessions separately
- 25% discount Monday-Thursday for hotel guests
- 16+ only

**Netherwood Hotel:**
- All guests must pay separately for Thermal Journey experience
- Daily access 10am-5:30pm
- Robes, towels and slippers included in Thermal Journey
- Adults only

**Appleby Manor:**
- All guests must pay separately for spa access
- Spa breaks available with overnight stays
- 16+ (some treatments require 18+)

## Usage Guidelines

### When Writing About Pricing
- Be specific when you have exact figures
- Use the access model information to help readers understand true costs
- Mention the tiered system for properties like Lodore Falls
- Note the discount structure at Low Wood Bay

### When Comparing Spas
- Include access model as a comparison factor
- Point out when "cheaper" rooms may cost more overall due to spa charges
- Highlight the value of included access properties

### Example Usage

Good: "At Armathwaite Hall, spa access is included with every room. At Lodore Falls, suites get two hours free per night while standard rooms pay £35 per person for a 2-hour session."

Good: "Low Wood Bay charges separately for spa access, though hotel guests get 25% off Monday to Thursday."

Good: "Netherwood Hotel requires all guests to pay separately for the Thermal Journey experience, which runs daily from 10am-5:30pm."
